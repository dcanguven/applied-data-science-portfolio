A collection of independent data science notebooks covering exploratory analysis, predictive modeling, forecasting, feature engineering, visualization, and data quality investigation.

Each notebook in this repository is structured as a standalone analytical project. The focus is on clear workflows, readable analysis, and practical problem-solving using real datasets.

The notebooks are written in a simple and direct style, with an emphasis on clarity, structured thinking, and practical interpretation of results.

This repository is designed as a professional portfolio of applied work rather than a tutorial series or academic study collection.

## Approach

Each notebook in this repository starts with a specific question, problem, or unusual pattern in the data.

Some projects focus on identifying hidden data quality issues, investigating unrealistic pricing behavior, comparing modeling approaches, or exploring relationships between variables to better understand the structure of the dataset.

The goal is not only to build models, but also to understand what the data is actually saying before making predictions or drawing conclusions.

The notebooks focus on:

- structured problem-solving
- exploratory analysis
- data quality investigation
- anomaly and pattern detection
- feature relationship analysis
- practical interpretation of results
- evaluation of different modeling approaches
- reliable predictive workflows using real datasets

Additional notebooks and projects will continue to be added over time.
